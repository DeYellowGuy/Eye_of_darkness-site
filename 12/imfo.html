MP, LVL UPS and more

So MP are really important in this game. DO NOT WASTE THEM! You can use MP to
  • Use magic - battle
  • Double jump and dash - stages
  • Destroy enemies attacks - battle / uses MAXMP
About LVL UPS, I can say... this is random. ONLY LUCK!
NORMAL MODE
Can upgrate HP, MP, ATK, DEF by 1 to 3, ONLY LUCK!
HARD MODE
Can upgrate HP, MP, ATK, DEF by -1 to 3, so you can eve LOSE points!

Different modes

As you saw at previous paragraph, yes, there are 2 different modes - Normal and HARD. Lets talk about difference
NORMAL MODE
  • Attacks from enemies are on normal speed
  • Already told about LVL up "1 to 3"
  • If you died you can restart from last checkpoint
NORMAL MODE
  • Attacks from enemies are faster and "deadly things" are doubled
  • Already told about LVL up "-1 to 3"
  • If you died, game over! you need to start AGAIN!
